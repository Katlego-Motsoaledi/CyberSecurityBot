# Cybersecurity Awareness Bot for South Africa 🇿🇦

![CI Status](https://github.com/Handsomebranden/CyberSecurityBot/actions/workflows/dotnet.yml/badge.svg)

A chatbot to educate South African citizens about cybersecurity threats.

## Features
- Voice greeting on startup
- South African-themed ASCII art
- Input validation system
- Localized cybersecurity advice
